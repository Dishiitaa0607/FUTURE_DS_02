
 Overview:

This project transforms raw marketing campaign data into actionable insights. Using Power BI, it visualizes key metrics to evaluate campaign success, optimize marketing strategies, and improve ROI.

- Features

Cleaned and transformed raw campaign data for accurate analysis

Calculated key marketing KPIs: Response Rate, ROI, Cost per Response, Total Spend, Customer Metrics

Analyzed performance across product categories, education levels, and marital status

Interactive Power BI dashboard with filters, trend charts, and KPI cards

Provides actionable insights for stakeholders and marketing teams
